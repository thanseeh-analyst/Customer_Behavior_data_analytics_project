# Customer Behavior Data Analyst Portfolio Project

## 📌 Project Overview

This project demonstrates a complete, industry-standard end-to-end data analytics workflow designed to simulate real-world business analysis responsibilities. The project covers the full analytics lifecycle, including data cleaning, transformation, SQL-based analysis, dashboard development, and business insight generation.

The objective of this project is to transform raw customer sales data into meaningful business intelligence that supports strategic decision-making and improves customer understanding.

---

## 🚀 Workflow Summary

### ✅ Data Cleaning & Preparation (Excel)

- Cleaned and transformed raw sales data using Microsoft Excel
- Handled missing values, formatting inconsistencies, and duplicate records
- Structured the dataset for analytical processing
- Exported the cleaned dataset into `.CSV` format for SQL and Power BI analysis

### ✅ Data Analysis (SQL)

- Performed business-oriented data analysis using SQL queries
- Analyzed customer purchase behavior, sales performance, and customer segments
- Generated insights related to customer loyalty, revenue trends, and purchasing patterns
- Simulated real-world analytical queries commonly used in business environments

### ✅ Visualization & Dashboard Development (Power BI)

- Developed an interactive Power BI dashboard for stakeholder reporting
- Created visualizations for:
  - Sales trends
  - Customer segmentation
  - Product performance
  - Revenue analysis
  - Customer purchasing behavior
- Enabled data-driven decision-making through interactive reporting

### ✅ Reporting & Business Insights

- Generated business insights and recommendations from analytical findings
- Summarized key patterns and trends identified from customer behavior data
- Demonstrated how analytics can support business strategy and operational improvement
